# Prompt-Engineering-Week1

# Week 1 - Prompt Basics: Zero-Shot vs Few-Shot

## Objective

Compare zero-shot and few-shot prompting by classifying customer support messages into:

- Complaint
- Question
- Praise

## AI Tools Used

- ChatGPT
- Claude
- Gemini

## Repository Contents

- Zero-shot prompt
- Few-shot prompt
- Results comparison
- Screenshots

## Summary

ChatGPT classified all 10 messages correctly in both prompts.

Claude and Gemini classified 9 out of 10 correctly, labeling the final message as a Complaint instead of a Question.

The experiment showed that while few-shot prompting did not improve accuracy on this simple dataset, it provides better guidance and is expected to perform better on more complex tasks.
